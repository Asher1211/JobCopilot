## 工作内容

### 1. 数据治理与清洗

负责大众点评、美团外卖、抖音本地生活、高德、京东、淘闪购等本地生活平台的数据清洗、整理与入库，通过飞书 Base 连接器完成数据接入。

### 2. Agent研发与优化

参与零售运营与数字运营 Agent 的迭代优化，包括：

- 意图路由（Intent Routing）
- 多 Agent 协作（Multi-Agent）
- 记忆系统更新（Memory）

### 3. 投放Agent建设

搭建本地生活平台广告投放 Agent，将业务经验沉淀为 AI 决策规则，实现自动化运营。

### 4. LLM知识工程

参与 LLM Wiki 知识工程建设，整理知识体系，构建知识库，参考 Karpathy 方法论开展相关工作。

### 5. AI竞赛支持

配合外部顾问参与飞书效率先锋赛事相关工作。
